# 📖 OIO English

Output → Input → Output — 每天 25-35 分钟，坚持说英语

一个帮助练习英语口语的 Android App，遵循 **OIO（Output → Input → Output）** 学习循环。使用 Kotlin + Jetpack Compose 开发。

---

## 📥 导入学习计划

准备好 xlsx 文件（列名：**Day | Topic | O1 | I(Input) | O2 | Anki**），在 Today 或 Plan 页面点击右下角 **+** 按钮选择文件。

| 列 | 说明 |
|---|---|
| Day | `第6天` / `Day 6` / `6` |
| Topic | 话题名 |
| O1 | 先自己试试说的内容 |
| I(Input) | 关键短语和句型 |
| O2 | 组合输出的完整句子 |
| Anki | `中文场景 \| English sentence`（竖线分隔，每行一张卡） |

> 💡 同一天 + 同话题名 → 自动合并内容。重新导入同一天 → 覆盖旧内容。

导入前会显示预览确认。

## ⏱️ 每日流程（25-35分钟）

### 🔥 热身 — 回顾前一天的 Anki（2分钟）

快速过一遍前一天的 Anki 卡片，看中文说英文。卡住的记下来今天重点学。

> 在 **Plan** 页面找到前一天 → 点右边 **→** 跳到 Today 直接复习

### 🅾️ O1 — 先自己试试（1-2分钟/话题）

点开 O1 卡片，看到引导问题。**硬着头皮说**——不会说也要说，卡住的地方就是今天要学的。

> 自己说就行，不用工具。卡住的地方记下来，下一步去问 AI。

### 🅸 I — 学语言块（2-3分钟/话题）

打开 I 卡片，把 O1 卡住的地方拿去问 AI：
- "这句话用英语怎么说？"
- "给我 3 个更地道的说法"

把 AI 的答案记下来，点 🔊 Read 听发音。

> 可以用豆包 / Kimi / ChatGPT。

### 🅾️ O2 — 组合输出（2分钟/话题）

用刚学到的语言块，把完整句子说出来。**至少说 2 遍**，直到顺口为止。

> 播放录音时可以调 **0.75x** 慢放，更容易听出问题。

### 🌙 睡前 Anki（3-5分钟）

看中文想英文 → 点击翻转核对 → 点 Next → 全部过完后点 **✅ Mark Complete**。

```
正面：在家里住
背面：I live in an apartment
操作：👆 点击翻转 → [Next →]
```

> 完成后卡片不会消失，随时可以回来复习。

## 📋 Today — 日常学习

选择 Day 序号后，每个话题显示为 4 张隐藏卡片（O1 / I / O2 / Anki），点击展开内容。

| 操作 | 说明 |
|---|---|
| 👆 点击卡片 | 展开/收起 |
| 🎤 Rec / ⏹ Rec | 开始/停止录音 |
| ▶️ Play / ⏹ Stop | 播放/停止录音 |
| 0.75x / 1x / 1.25x | 播放速度切换 |
| 🔊 Read / ⏹ stop | TTS 朗读/停止 |
| ✅ Mark Complete | 标记该环节完成 |
| 🏆 Easy/Medium/Hard | 话题难度标记 |
| ✏️ 长按内容 | 编辑文字 |

---

## 📅 Plan — 周计划

- **已完成的话题** → 绿色背景，点击可展开 O2
- **未完成的话题** → 粉色背景，只显示话题名
- **→ 按钮** → 跳到 Today 学习该天
- **复习日 / 模拟考** → 按难度颜色显示话题（绿=Easy / 橙=Medium / 红=Hard）

## 🔊 TTS 朗读

国产手机（小米/OPPO/华为等）需要先下载英语语音包：

1. 设置 → 更多设置 → 文字转语音(TTS)输出
2. 点当前引擎旁边的 ⚙️ → 下载英语(美国)语音数据
3. 下载完成后回来点 🔊 Read

朗读时点 **⏹ stop** 可随时停止。

## 📝 每日反思

```
gradlew assembleRelease  # 发布 APK
```
